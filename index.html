Questions and Answers

1. What type of request do I need to make to add a new resource
via an API? Answer: Push

2. CSS - if we want to align using flex-box in the middle on the
horizontal axis, what are the flex properties i need to use?
3 answers needed. Answers: Align-items, align-content, display flex

3. JS - what is a function parameter? Answer: Additional instructions added to a function that must be met
in order for the function to work.

4. JS - What is an event-listener? Answer: Event-listeners are a type of method that causes a response when a
particular action is carried out on the front-end server, such as a
mouse click.

5. What do we mean by the DOM? Answer: the DOM stands for Document Object Model, and is the basic structure
from which online documents are created.

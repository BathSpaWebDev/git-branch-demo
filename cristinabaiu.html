ANSWERS to QUESTIONS

Questions:

1) What type of http request do I need to make to add a new resource via a API?
2)CSS, if we want to align using flex-box in the middle on the horizontal axis what are the flex properties I need to use ?
3)JS, what is a function parameter?
4)JS, what is an event listener?
5)What do we mean by the DOM?

Answers:

1)We use HTTP POST request to add a new resource.
2)To align items in the middle on the horizontal axis we use:

'display: flex;'
'align-items: center;'
'justify-content: center;'
3)A function parameter allows us to perform tasks or calculate a value.
4)An event listener is a way of making a button or a part of the page to respond to the users actions.
5)DOM or Document Object Model is a programming interface for web documents.
